## Installation

First clone the repo:
```
git clone https://github.com/Mystic2103/Mystic-Machine-PU.git
```
After cloning, run:
```
npm install
```
To launch, run:
```
node .
```

### Bot Token
- You can change the bot token in: <br>
`./config/index.js` <br>
